# Changelog

## 0.3.2

- published the refreshed Marketplace README and short description
- kept release automation aligned for the next tagged release

## 0.3.1

- changed the Marketplace display name to `Project Launcher Buttons` to avoid a naming collision during first publication
- hardened the release workflow for manual tag dispatches and Marketplace publishing

## 0.3.0

- replaced the old linear task-label setup with a friendlier launcher setup menu
- added quick setup with automatic recommended task detection
- made configured start, stop, and health bindings editable directly from the sidebar
- added setup readiness status to make missing configuration obvious
- updated command contributions to use icons so the panel toolbar is less cluttered

## 0.2.0

- added a dedicated Project Launcher activity-bar container
- added sidebar Control Panel and Activity views
- added output channel logging for task lifecycle events
- added refresh, show output, and clear activity commands
- added repository-ready CI and release workflow templates for extracting the extension into a separate repository

## 0.1.0

- added product-grade extension metadata for VSIX packaging
- added optional health button and health task command
- added duplicate-start protection
- improved restart flow with stop-wait-start sequencing
- improved task picker details with task source and workspace folder
- added packaging-friendly manifest files list, repository and license
- added extension icon and marketplace-style branding metadata
